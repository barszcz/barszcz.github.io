Work

Here are some projects I've worked on:

BeatSoup

A single-page social music-streaming app inspired by SoundCloud, built primarily with Backbone.js, consuming a RESTful JSON API served by a Ruby on Rails backend. Features a persistent now-playing queue, user uploading, playlists (which may be public or private), comments and tags. Uses the Paperclip gem in conjunction with Amazon AWS to handle file upload and storage, implements a CompositeView class in Backbone to better modularize code, and overrides Backbone.sync to facilitate global error handling. (Github)

JSteroids

A vaguely psychedelic browser-based version of the classic arcade game Asteroids, powered by JavaScript and HTML5 Canvas. Pretty colors! Lasers! Space! Cats! What more could you possibly want?? For this project I extended JavaScript's Array prototype to add functions for vector math, and wrote an "inherits" function to let me use JS's prototypal inheritance in a more classical manner. These two decisions enabled a huge amount of code reuse.(Github)

ActiveRecord Lite

A bare-bones reimplementation of Ruby on Rails' ActiveRecord ORM. Takes full advantage of Ruby's powerful reflection and metaprogramming capabilities in order to implement a domain-specific language, which gets translated into raw SQL queries, on top of Ruby. In its current form it implements search queries, insert/update, and associations.
