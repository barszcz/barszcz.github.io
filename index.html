Work

Here are some of my recent projects:

BeatSoup (Live | Github)

A single-page social music-streaming app inspired by SoundCloud.

  • Built primarily with Backbone.js, consuming a RESTful JSON API served by a Ruby on Rails backend.
  • Features a persistent now-playing queue, user uploading, playlists (which may be public or private), comments and tags.
  • Uses the Paperclip gem in conjunction with Amazon AWS to handle file upload and storage.

Fixpoint (Live | Github)

A reader for Hacker News built with React.js.

  • Leverages React's fast virtual-DOM architecture to create a highly performant single-page app.
  • Consumes Hacker News' Firebase API, allowing for easy access to real-time updates.
  • Also uses react-mini-router to handle routing and Moment.js for generating human-friendly timestamps.

JSteroids (Live | Github)

A vaguely psychedelic browser-based version of the classic arcade game Asteroids.

  • Written in JavaScript and uses the HTML5 <canvas> API for animation.
  • Pretty colors! Lasers! Space! Cats! What more could you possibly want??
  • Extends JavaScript's Array prototype to add functions for vector math.
  • Uses an "inherits" function to let me use JS's prototypal inheritance in a more classical manner.

ActiveRecord Lite (Github)

A basic reimplementation of Ruby on Rails' ActiveRecord ORM library.

  • Takes advantage of Ruby's powerful reflection and metaprogramming capabilities.
  • Implements a domain-specific language, which gets translated into raw SQL queries, on top of Ruby.
  • Currently implements search queries, insert/update, and associations.
